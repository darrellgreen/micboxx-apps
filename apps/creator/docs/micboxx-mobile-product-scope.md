# MicBoxx Mobile Product Scope

Date: 2026-04-08

## Product framing

MicBoxx mobile is the native listening and discovery client for the MicBoxx platform. It is not a separate product and it should not invent parallel business rules.

## Launch posture

- Listener-first UX
- Creator-compatible account model
- Same Drupal source of truth
- Same public and dashboard contracts where they are already stable

## Phase 1

- Auth and session bootstrap
- Home and discover surfaces
- Search
- Track detail
- Album detail
- User profile detail
- Playlist detail
- Queue and player shell
- Library basics

## Phase 2

- Comments
- Follows
- Notifications
- Messaging
- Purchase and subscription access handling

## Phase 3

- Light creator utilities
- Upload entry points
- Analytics snapshots
- Release management basics

## Non-goals for the initial build

- Mobile-specific backend
- Dashboard parity
- Mixed-cart commerce
- Creator-heavy admin workflows
- Rewriting server or web business logic inside the app
